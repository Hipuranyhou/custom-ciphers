# custom-ciphers
Custom tools for basic ciphers.

### Rot13
```
rot13 STRING
```

### Caesar
```
caesar -e NUMBER/KEY STRING
caesar -d NUMBER/KEY STRING
```
Bruteforce mode:
```
caesar -b STRING
```
