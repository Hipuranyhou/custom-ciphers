# custom-ciphers
Custom tools for basic ciphers.

### Rot13
Prints only rotated string on newline.

Returns 0 on success, 1 on error.
```
rot13 STRING
```

### Caesar
Prints only shifted string on newline.

Returns 0 on success, 1 on error.
```
caesar NUMBER STRING
```
Bruteforce mode:
```
caesar -b STRING
```
